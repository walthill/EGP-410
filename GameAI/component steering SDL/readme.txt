[TO COMPILE]
**Visual Studio required**

- Unzip files
- Run the steering solution file (.sln)
- Build with F7
- Run in Debug Mode with F5

[CONTROLS]
Left Mouse - Set movement waypoint for black arrow
Enter Key - Spawn red arrow
D Key - Delete random red arrow
Escape Key - Quit
