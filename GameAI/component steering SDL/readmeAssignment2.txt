[TO COMPILE]
**Visual Studio required**

- Unzip files
- Run the steering solution file (.sln)
- Build with F7
- Run in Debug Mode with F5

[CONTROLS]
WEIGHT CONTROLS
1 - Wander Up
2 - Wander Down
3 - Group Align Up
4 - Group Align Down
5 - Cohesion Up
6 - Cohesion Down
7 - Separation Up
8 - Separation Down

RADIUS CONTROLS
Q -Separation Up
W - Separation Down
E - Group Align Up
R - Group Align Down
T - Cohesion Up
Y - Cohesion Down

Enter Key - Spawn 10 red arrows
D Key - Delete random red arrow
Escape Key - Quit
